# ⚽ VarzeaOnline

Um sistema web completo para organizar e gerenciar campeonatos de **FIFA / EA FC** entre amigos, com suporte a múltiplos formatos de torneios, automação de rodadas e classificação em tempo real.

---

## 📋 Sobre o Projeto

O **VarzeaOnline** é uma aplicação web desenvolvida em **ASP.NET MVC** que facilita a criação e gerenciamento de campeonatos de FIFA.

Com uma interface inspirada no estilo da EA Sports, o sistema permite desde a criação do campeonato até o acompanhamento completo das rodadas e classificação.

---

## ✨ Destaques

- 🎮 **50+ Times Reais** — Premier League, La Liga, Série A e mais  
- 🏆 **Múltiplos Formatos** — Pontos Corridos, Mata-Mata, Grupos + Mata-Mata  
- 📊 **Classificação Automática** — Atualização em tempo real  
- 👥 **Sistema de Usuários** — Login e cadastro  
- 🎨 **Dark Theme** — Visual moderno estilo EA FC
